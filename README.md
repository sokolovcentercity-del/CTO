# Проверка документов — AI Анализ

Standalone веб-приложение для автоматической проверки комплектов документов.
Работает на любом PHP-сервере. **Не требует** miniapps.ai.

---

## Установка

### 1. Скопируйте файлы на сервер

```
doc-checker/
├── index.html
├── main.js
├── styles.css
├── doc-analyzer.js
├── pdf-processor.js
└── api/
    ├── config.php       ← ВСТАВЬТЕ КЛЮЧ СЮДА
    ├── ai-proxy.php
    └── .htaccess
```

### 2. Вставьте API-ключ

Откройте `api/config.php` и замените:
```php
define('ANTHROPIC_API_KEY', 'ВСТАВЬТЕ_ВАШИ_КЛЮЧ_СЮДА');
```
на ваш ключ `sk-ant-...` с сайта **console.anthropic.com**.

### 3. Готово

Откройте `https://ваш-сайт.ru/doc-checker/` в браузере.

---

## Требования к серверу

- PHP 7.4+ с расширением **curl**
- HTTPS (рекомендуется)
- Apache / Nginx с поддержкой `.htaccess`

## Безопасность

- `api/config.php` защищён через `.htaccess` — недоступен снаружи
- API-ключ хранится только на сервере, в браузер не передаётся
- Для дополнительной защиты укажите в `config.php`:
  ```php
  define('ALLOWED_ORIGIN', 'https://ваш-сайт.ru');
  ```

## Поддерживаемые форматы

- **PDF** — текстовые и сканы (OCR через AI)
- **DOCX** — документы Word
- **XLSX** — таблицы Excel
- **TXT, CSV, MD** — текстовые файлы

---

## Типы проверок

**Этап 1 — Извлечение данных (AI):**
- Стороны, ИНН, КПП, ОГРН
- Номера, даты документов
- Позиции, количества, цены, суммы
- Ссылки на договор, подписи, печати

**Этап 2 — Программные проверки:**
- Несоответствие ИНН одной организации
- Даты в будущем, просроченные документы
- Нарушение хронологии (акт раньше договора)
- Арифметические ошибки в суммах
- Сумма документа превышает контракт
- Несовпадение ссылок на договор
- Отсутствие подписей и печатей
- Несоответствие товаров контракту
- Специфические проверки для претензий, сертификатов

**Этап 3 — AI-заключение (Claude Sonnet):**
- Оценка рисков
- Главные замечания
- Практические рекомендации

## Проблемы и решения

**"API key not configured"** — вставьте ключ в `api/config.php`

**"cURL error"** — проверьте доступ сервера к интернету, должен работать curl к `api.anthropic.com`

**PDF не читается** — убедитесь что PDF не защищён паролем

**OCR не работает** — проверьте баланс на console.anthropic.com
